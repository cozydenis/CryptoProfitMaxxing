Training Approach

Chronological Split — No Future Leakage

TRAIN (80%) — 252 days
TEST (20%) — 63 days
  • Strict time-order split — never shuffled
  • Prevents the classic ML-on-time-series mistake (future data leakage)
  • Same split for all 3 models — fair comparison
  • Target: did BTC go UP or DOWN the next day?

Ray Tune Hyperparameter Search

20 random configurations per model — every trial logged to MLflow
